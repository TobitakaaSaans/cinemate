Cinemate - Match With your next movie

ReactJS App to explore the OMDb API 
[http://ombdapi.com](http://ombdapi.com)

## Core Features

- Search movies by title
- View movie details 

## Bonus features 

- Tag a movie as favourite
- Redo a previous search by search history

## Used libraries

- Axios 
- Redux
- Styled components 
- React-materialize
- Animate.css
